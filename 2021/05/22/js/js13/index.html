Lorem-Ipsum

前言

  • Lorem Ipsum假文介紹 : https://ithelp.ithome.com.tw/articles/10130206

功能說明

  1. 標題
  2. 輸入區域 : 可以輸入0~9之間的數字x
  3. 顯示結果 : 產生出x段的假文

HTML重點

  • 標題

  • 輸入區域

    • 將整個form的Class設為lorem-form
    • 將輸入方格的id設為amount，並讓其預設顯示5

  • 顯示結果 :

    • 將整個顯示結果的Class設為lorem-text

JavaScript重點

  1. 將所有假文存入text陣列中

  2. 利用class="lorem-form"去抓取整個form

  3. 利用id="amount"去抓取輸入方格

  4. 利用class="lorem-text"去抓取顯示結果

  5. 預設輸入方格的值為5

  6. 為form新增提交事件

下列程式碼皆在form.addEventListener()之內

  1. 按了按鈕之後，停止提交表單

    • 因為Submit事件是一瞬間的事，若沒有使用preventDefault()，顯示結果將會顯示一下便消失
    • 詳請請閱讀下方參考資料

  2. 將輸入方格的值轉為數字

  3. 在索引值0~9之間隨機取一數

    • 詳情於【04】網頁練習 - 更改背景顏色中有提到

  4. 在索引值0~9之間隨機取一數

    • innerHTML、Array.map()與Array.join()於【11】網頁練習 - 分類型Menu(上)中有提到
    • Array.slice(begin, end) : 回傳一個新陣列，為原陣列選擇之begin至end（不含 end）的部分(原本的陣列將不會被修改)。

參考與引用來源

  1. preventDefault()介紹
  2. Array.slice()介紹
