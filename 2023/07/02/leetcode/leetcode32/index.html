【Leetcode解題】242. Valid Anagram

Posted on 2023-07-02 In Leetcode解題 , Easy , Grind 125 Views:
【Grind 125】 Week 1-7

題目

  • 題目連結
  • 目標函式 : isAnagram(s, t)
    • Input : s: str、t: str
    • Output : bool

解題概念

  • 題意 : 給予兩個字串 s、t，判斷 s 是否為 t 的 Anagram
  • Anagram (同字母異序字) : https://en.wikipedia.org/wiki/Anagram

我的寫法

  • 想法 : 使用 Hashmap 去處理
    1. 將 s 與 t 分別轉換為 {字元 : 字元個數} 的 Hashmap
    2. 比較兩個 Hashmap 是否相等

  • 成效 :
    • Time Complexity : $\color{DB4D6D}{O(n)}$
    • Runtime: 33 ms (faster than 86.67%)
    • Memory Usage: 13.6 MB (more than 99.63%)
