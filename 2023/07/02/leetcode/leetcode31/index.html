【Leetcode解題】226. Invert Binary Tree

Posted on 2023-07-02 In Leetcode解題 , Easy , Grind 125 Views:
【Grind 125】 Week 1-6

題目

  • 題目連結
  • 目標函式 : invertTree(root)
    • Input : root: TreeNode
    • Output : TreeNode

解題概念

  • 題意 : 給予一 Binary Tree 的 root，對其進行反轉並回傳 root
  • 此題是在操作 Binary Tree範疇中很經典的一題

我的寫法

  • 想法 : 使用遞迴的方式
    1. 若 root 為空，則直接回傳 root
    2. 否則交換 root.left 與 root.right
    3. 最後再回傳 root

  • 成效 :
    • Time Complexity : $\color{DB4D6D}{O(n)}$
    • Runtime: 21 ms (faster than 29.37%)
    • Memory Usage: 13.6 MB (more than 19.95%)
