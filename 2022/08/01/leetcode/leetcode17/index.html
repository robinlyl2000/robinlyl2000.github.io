解題概念

  • 此題簡單來說，就是要處理簡單物件導向的操作
    1. NumArray + 一陣列 nums : 建立一物件，並將 nums 放入其中
    2. update + index + val : 將 nums[index] 的值改為 val
    3. sumRange + left + right : 回傳從 nums[left] + nums[left + 1] + $\cdots$ + nums[right] 之總和
