【Leetcode解題】240. Search a 2D Matrix II

發表於2022-07-24|更新於2023-07-20|Leetcode解題Daily ChallengeMedium
|閱讀量:

題目

  • 題目連結

解題概念

  • 此題簡單來說，就是要從一陣列 matrix 中，找出目標 target
  • 實際執行上，要從陣列的特性下手 :
    1. 對 Each Row 而言，最大值都位於最右方
    2. 對 Each Column 而言，最小值都位於最上方
    3. 故可推論出，搜尋起始點為右上方
  • 接著，主要迴圈的判斷方式為 :
    • 若 matrix[i][j] = target : 找到目標，回傳 true
    • 若 matrix[i][j] > target : 代表 matrix[i][j] 那列左方還有更小的元素，故 j -= 1
    • 若 matrix[i][j] < target : 代表 matrix[i][j] 那排下方還有更大的元素，故 i += 1
